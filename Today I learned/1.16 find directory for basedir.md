# To print the current directory

```
$ python manage.py shell
```

```
>>> import os
>>> print(os.getcwd())
```

# To find the directory of `base_dir`

```
$ python manage.py shell
```

```
>>> from django.conf import settings
>>> print(settings.BASE_DIR)
```

We can also configure the BASE_DIR in settings.py too
