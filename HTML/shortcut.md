- `Ctrl + /` = `<!-- -->`
```html
<h1> hello </h1>
<!--This is a comment for html-->
```

- `any tag name + >` = `<opening tag> <closing tag>`
- `any tag name + tab` = `<opening tag> <closing tag>`

Type ```body>``` or `body` + `tab`

```html
<body>
  
</body>
```

Type ```head>``` or `head` + `tab`
```html
<head></head>
```
