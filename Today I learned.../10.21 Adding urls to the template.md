# Adding url link to the template

<em>home.html</em>
```html
<ul>
  <li><a href = "{% url 'home' %}">click home</a></li>
</ul>
```

This `home` is from app1/urls.py

app1/urls.py

```py
from django.urls import path
from . import views
urlpatterns = [
    path("",views.index, name="home")
]
```
